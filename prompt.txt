Saya sedang mengembangkan backend REST API menggunakan CodeIgniter 3.1.10 untuk sistem IT Service Desk dan monitoring perangkat.

Database MySQL sudah terhubung melalui application/config/database.php.

Saya memiliki tabel berikut:

tickets
users
messages

Struktur tabel messages:

- id (INT AUTO_INCREMENT PRIMARY KEY)
- ticket_id (INT)
- sender_type (ENUM: admin, device)
- sender_id (INT)
- message (TEXT)
- created_at (DATETIME)

Relasi:
messages.ticket_id → tickets.id
messages.sender_id → users.id (jika sender_type = admin)

Saya ingin membuat API untuk sistem percakapan pada ticket (ticket chat).

Gunakan struktur CodeIgniter:

Controller:
application/controllers/api/Messages.php

Model:
application/models/Message_model.php

Gunakan Query Builder CodeIgniter.

---

API Endpoint yang harus dibuat

1️⃣ GET /api/tickets/{ticket_id}/messages

Mengambil semua pesan pada ticket tertentu.

Data yang dikembalikan:

- message_id
- sender_type
- sender_name
- message
- created_at

Jika sender_type = admin,
ambil nama dari tabel users.

Jika sender_type = device,
gunakan reporter_name dari tabel tickets.

Response contoh:

{
  "status": "success",
  "message": "messages retrieved",
  "data": [
    {
      "id": 1,
      "sender_type": "device",
      "sender_name": "Budi Santoso",
      "message": "Komputer sering mati sendiri",
      "created_at": "2026-03-11 09:30:00"
    },
    {
      "id": 2,
      "sender_type": "admin",
      "sender_name": "Admin IT",
      "message": "Baik, kami akan cek segera.",
      "created_at": "2026-03-11 09:32:10"
    }
  ]
}

---

2️⃣ POST /api/tickets/{ticket_id}/messages

Digunakan untuk mengirim pesan pada ticket.

Body JSON contoh:

{
  "sender_type": "admin",
  "sender_id": 2,
  "message": "Kami sedang mengecek komputer tersebut."
}

Jika sender_type = device,
sender_id bisa NULL.

Simpan pesan ke tabel messages.

created_at harus otomatis menggunakan waktu sekarang.

Response:

{
  "status": "success",
  "message": "message sent"
}

---

Tambahkan routing di:

application/config/routes.php

$route['api/tickets/(:num)/messages'] = 'api/messages/index/$1';
$route['api/tickets/(:num)/messages/send'] = 'api/messages/send/$1';

---

Gunakan format response JSON standar:

Success

{
  "status": "success",
  "message": "text",
  "data": []
}

Error

{
  "status": "error",
  "message": "text"
}

Pastikan kode kompatibel dengan CodeIgniter 3.1.10 dan dapat langsung diuji menggunakan Postman.
